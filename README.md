**EJERCICIO**  
Dada una app de notas subida al repositorio, realiza las siguientes actividades:  
Ejercicio 1. crea 2 issues en el repositorio con la siguiente información:  
    - Títulos:  
              1. "Implementar búsqueda de notas"  
              2. "Bug: La aplicación se cierra al borrar una nota"  
    - Una descripción  
    - Agrega al menos dos labels   
    - Asignarlo a un integrante del proyecto  
Ejercicio 2. crea un Project Board con formato Kanban. Luego:  
  - Añadí los issues creados al tablero  
  - Simulá el flujo de trabajo moviendo los issues a través de las distintas columnas
